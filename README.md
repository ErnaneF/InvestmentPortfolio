# Investment Portfolio Analyzer

A tool to analyze investment portfolios and individual stocks.

## 🎯 Goals
- This project aims to analyze real or hypothetical investment portfolios to help investors make informed decisions about which stocks to buy.
- It will provide insights for beginner and intermediate investors.

## 🌱 Future Goals
- Transform this project into a local app with a user interface.

## 📚 Data Sources
- [yfinance](https://pypi.org/project/yfinance/) – Yahoo Finance Python library for historical stock data
- [StatusInvest](https://statusinvest.com.br) – Manual data collection for specific information
- [Fundamentus](https://www.fundamentus.com.br) – Financial data and company metrics

## 🛠️ Technologies and Tools
- Python
- pandas
- yfinance
- matplotlib
- numpy
- Jupyter Notebook
